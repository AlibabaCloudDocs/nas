# 错误代码表 {#concept_62603_zh .concept}

NAS API 调用出现问题时，会返回各种错误码信息。

## 错误码组成 {#section_yr5_jjg_hfb .section}

错误码由以下4部分组成：

|组成部分|释义|
|----|--|
|HttpCode|HTTP 响应码，符合 HTTP 协议语义|
|Code|错误标识|
|Message|错误消息|
|Advice|建议处理方式|
